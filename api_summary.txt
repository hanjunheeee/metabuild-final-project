API 상세 정리 (Postman용)
Base URL: http://localhost:7878

[Analytics]
- GET http://localhost:7878/api/analytics/trends/keywords
  기능: 검색 키워드 트렌드 목록 조회
  Body: none
  Postman: GET 선택 후 Send

- GET http://localhost:7878/api/analytics/trends/purchase
  기능: 구매 관련 트렌드 목록 조회
  Body: none
  Postman: GET 선택 후 Send

- GET http://localhost:7878/api/analytics/trends/library
  기능: 도서관 대출 트렌드 목록 조회
  Body: none
  Postman: GET 선택 후 Send

- POST http://localhost:7878/api/analytics/log/search
  기능: 검색 로그 저장 (키워드/도서명)
  Body(JSON): {"keyword":"string","bookTitle":"string"}
  Postman: Body -> raw -> JSON 선택 후 위 JSON 입력

- POST http://localhost:7878/api/analytics/log/action
  기능: 도서 행동 로그 저장 (구매조회/도서관조회 등)
  Body(JSON): {"bookId":123,"actionType":"PURCHASE_VIEW"}
  Postman: Body -> raw -> JSON

- GET http://localhost:7878/api/analytics/blocked-keywords
  기능: 차단 키워드 목록 조회
  Body: none

- POST http://localhost:7878/api/analytics/blocked-keywords
  기능: 차단 키워드 추가
  Body(JSON): {"keyword":"string"}

- DELETE http://localhost:7878/api/analytics/blocked-keywords/{keyword}
  기능: 차단 키워드 삭제
  Body: none
  Postman: URL에 {keyword} 실제 값 입력

[Books]
- GET http://localhost:7878/api/books
  기능: 전체 도서 목록/검색
  Query: http://localhost:7878/api/books?query=
  Body: none

- GET http://localhost:7878/api/books/{id}
  기능: 도서 상세 조회
  Body: none

- GET http://localhost:7878/api/books/{id}/summary
  기능: 도서 AI 요약 조회
  Body: none

- GET http://localhost:7878/api/books/{id}/shops
  기능: 도서 가격/서점 정보 조회
  Query: http://localhost:7878/api/books/{id}/shops?title=
  Body: none

- GET http://localhost:7878/api/books/bestsellers/aladin
  기능: 알라딘 베스트셀러 TOP10 조회
  Body: none

- GET http://localhost:7878/api/books/bestsellers/kyobo
  기능: 교보 베스트셀러 TOP10 조회
  Body: none

- GET http://localhost:7878/api/books/bestsellers/yes24
  기능: YES24 베스트셀러 TOP10 조회
  Body: none

- GET http://localhost:7878/api/books/loans/seoul
  기능: 서울 월간 대출 TOP10 조회
  Body: none

- POST http://localhost:7878/api/books
  기능: 도서 등록 (관리자용)
  Body(JSON): BookDTO
  예시: {"bookId":1,"isbn":"string","title":"string","author":"string","publisher":"string","publishedDate":"2024-01-01","summary":"string","imageUrl":"string","ages":"string","tag":"string","borrowedAmount":0}

- PUT http://localhost:7878/api/books/{id}
  기능: 도서 수정 (관리자용)
  Body(JSON): BookDTO

- DELETE http://localhost:7878/api/books/{id}
  기능: 도서 삭제 (관리자용)
  Body: none

- GET http://localhost:7878/api/users/{id}

  Body: none

- GET http://localhost:7878/api/users/check-nickname

  기능: 비밀번호 변경(마이페이지)

- PATCH http://localhost:7878/api/users/{id}/status
